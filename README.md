
# 🎬 Leonardo DiCaprio Filmography Analysis Dashboard

## 📖 Project Overview

This project is an interactive Power BI dashboard that analyzes Leonardo DiCaprio's filmography using movie-related data. The dashboard provides insights into box office performance, profitability, IMDb ratings, genres, release years, and collaborations with directors.

The goal of this project is to demonstrate data visualization, business intelligence, and dashboard design skills using Power BI.

---

## 📊 Dashboard Features

- 🎥 Total Movies
- 💰 Total Revenue
- 📈 Total Profit
- ⭐ Average IMDb Rating
- ⏱ Average Runtime
- 📊 Top 10 Highest Grossing Movies
- 💵 Most Profitable Movies
- 📉 Budget vs Revenue Analysis
- 🍿 Genre Distribution
- ⭐ IMDb Rating by Release Year
- 🎬 Director Collaboration Analysis

---

## 📌 Key Insights

- Titanic is Leonardo DiCaprio's highest-grossing movie.
- Martin Scorsese is his most frequent collaborator.
- Drama is the most common genre in his filmography.
- Leonardo DiCaprio's movies maintain a consistently high IMDb rating.
- Several high-budget films generated exceptional profits.
